Help & Instructions

Save/Load Project

  • To load a project from file, click the Choose file button on top-right of the screen and choose the model file in JSON. Click Open.
    Click on Load project button to load the project to canvas.
  • To save the current project into a JSON file, click the Save project button on top-right of the screen. The project will be saved as untitled.json. You can rename the file afterwards.

Drawing Shapes

  • To draw a shape, click one of the available shapes (square, triangle, or line) on the toolbox. Click on the canvas to draw shape at your cursor position.
  • Click on the shape button once again to toggle it off (this will enable dragging and resizing).

Dragging and Resizing

  • To drag an object, make sure you have toggled off any draw shape button.
  • Make sure you are on dragging mode by clicking on the Drag button in the toolbox.
  • To resize object, switch to resize mode by clicking on the Resize button.

Changing Color

  • To change the color of an object, select the object on the canvas by clicking on it.
  • Choose any of the colors available on the Colors toolbox.
